DISCARD INIT LEADERS

Condizioni:

-discard
Null Condition -> La carta ricevuta è null
Not in Hand Condition -> La carta ricevuta non fa parte di quelle del player

CHOOSE INIT RES

Condizioni:

-chooseResource
Null Condition -> La risorsa è null
Blank or Faith Condition -> La risorsa selezionata è di un tipo non accettabile
Excessive Amount -> è stato inviato un oggetto con un valore superiore a quello consentito

MANAGE WAREHOUSE

Condizioni:

-swapLines
Same Line -> Gli indici sono uguali, si sta tentando di scambiare una linea con sè stessa
Negative Line -> Indice di riga negativo, impossibile accedere all'array
Over Size -> L'indice eccede il massimo valore utilizzabile per accedere all'array

-addFromExtra
Null -> La risorsa passata è null
Negative Line -> Indice di destinazione negativo, impossibile accedere all'array
Over Size -> Indice oltre il valore massimo, impossibile accedere all'array
Extra Empty -> Controlla che il pool extra non sia vuoto
Not in Extra -> La risorsa non è presente nella lista di risorse extra. Invalida anche la selezione di faith e blank
Not Same Resource -> Si sta cercando di aggiungere alla riga nel deposito una risorsa di diverso tipo
Line Full -> Si sta cercando di aggiungere una risorsa a una linea del deposito già piena

-removeFromDepot
Negative Line -> Index negativo, impossibile accedere all'array
Over Size -> Index superiore al valore massimo, impossibile accedere all'array
Line Empty -> Si sta cercando di rimuovere una risorsa da una linea del deposito vuota

-depotToCard
Invalid State -> Si sta provando a mettere risorse nelle carte dal default. L'inserimento nelle carte è ammissibile solo dopo aver preso risorse
Line Empty -> Si sta provando a rimuovere una risorsa da una linea del deposito vuota
Not in Hand -> Viene controllata la presenza prima del tipo di abilità (DEPOT) e se questo va a buon fine, viene controllato il tipo di risorsa ammissibile
Card Inactive -> Il player ha la carta giusta a questa operazione, ma la carta non è stata attivata
Card Full -> Il deposito interno della carta è pieno 

-extraToCard
Invalid State -> Si sta provando a mettere risorse nelle carte dal default. L'inserimento nelle carte è ammissibile solo dopo aver preso risorse
Extra Empty -> Controlla che il pool extra non sia vuoto
Not in Extra -> La risorsa selezionata non è nelle extra resource. Invalida anche blank e faith
Not in Hand -> Viene controllata la presenza prima del tipo di abilità (DEPOT) e se questo va a buon fine, viene controllato il tipo di risorsa ammissibile
Card Inactive -> Il player ha la carta giusta a questa operazione, ma la carta non è stata attivata
Card Full -> Il deposito interno della carta è pieno
